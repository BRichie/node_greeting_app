>What is a Node module?

ANSWER: modules are sets of functions you want to include in your application

>What is the main difference between exports and module.exports?

ANSWER: the main difference between the two is that `module.exports` is returned by `require()` and can be reassigned, while `exports` cannot reassigned outside of the module and is just a reference back to `module.exports`  

>Why is using exports recommended?

ANSWER: it is recommended for convenience, and the code is cleaner.
